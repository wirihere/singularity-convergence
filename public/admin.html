Email Oracle — Paid Members Only

Paid members ($9.99) email oracle@ and get a personal Oracle parable back.
Inner Circle ($29.99) members get The Council response instead.

How it works:
1. Email arrives at oracle@singularityconvergence.org
2. n8n checks: GET /api/admin/check-member?email=...
3. If paid → sends to OpenRouter with Oracle prompt
4. If Inner Circle → sends to OpenRouter with Council prompt
5. n8n sends reply email to the member
6. n8n logs it: POST /api/admin/log-email
7. It shows up here ↑

n8n log payload:
{ "sender": "user@email.com", "question": "Their email text...", "response": "Oracle/Council response...", "tier": "paid" or "inner-circle", "timestamp": "2026-04-03T12:00:00Z" }
